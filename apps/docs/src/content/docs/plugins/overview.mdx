# Using Plugins

Learn how to effectively use plugins in your Brain Framework applications.

## Installing Plugins

<Tabs>
  <TabItem label="Official Plugins">

## Basic Usage

```javascript
import { Brain } from 'brain-framework';
import { DataNormalizer, FeatureAnalyzer } from 'brain-plugins';

const brain = new Brain();

// Add plugins
brain.use([
  new DataNormalizer({
    scale: [0, 1]
  }),
  new FeatureAnalyzer({
    features: ['mean', 'variance']
  })
]);
```

## Plugin Configuration

```javascript
// Configure plugin with options
const plugin = new DataNormalizer({
  scale: [0, 1],
  mode: 'minmax',
  precision: 2
});

// Add configured plugin
brain.use(plugin);

// Update configuration
plugin.configure({
  scale: [-1, 1]
});
```

## Plugin Management

```javascript
// Check if plugin exists
if (brain.hasPlugin('DataNormalizer')) {
  // Get plugin instance
  const plugin = brain.getPlugin('DataNormalizer');
  
  // Disable plugin temporarily
  plugin.disable();
  
  // Re-enable plugin
  plugin.enable();
  
  // Remove plugin
  brain.removePlugin('DataNormalizer');
}
```

## Using Multiple Plugins

```javascript
import { 
  DataNormalizer, 
  FeatureExtractor,
  PatternAnalyzer 
} from 'brain-plugins';

// Add multiple plugins with specific order
brain.use([
  new DataNormalizer(),  // Runs first
  new FeatureExtractor(), // Runs second
  new PatternAnalyzer()  // Runs last
]);

// Process data through all plugins
const result = await brain.process(data, {
  usePlugins: ['DataNormalizer', 'FeatureExtractor', 'PatternAnalyzer']
});
```

## Plugin Events

```javascript
// Listen to plugin events
plugin.on('processing', (data) => {
  console.log('Processing data:', data);
});

plugin.on('error', (error) => {
  console.error('Plugin error:', error);
});

// Global plugin events
brain.on('pluginAdded', (plugin) => {
  console.log('New plugin added:', plugin.name);
});
```

## Best Practices

1. **Order Matters**
```javascript
// Correct order: Normalize before analyzing
brain.use([
  new DataNormalizer(),
  new DataAnalyzer()
]);
```

2. **Error Handling**
```javascript
try {
  await brain.process(data);
} catch (error) {
  if (error.pluginName) {
    console.error(`Plugin ${error.pluginName} failed:`, error);
  }
}
```

3. **Resource Management**
```javascript
// Cleanup when done
await brain.cleanup(); // This will cleanup all plugins too
```

## Common Patterns

### Sequential Processing
```javascript
const result = await brain.process(data, {
  sequence: [
    ['DataNormalizer', { scale: [0, 1] }],
    ['FeatureExtractor', { features: ['mean'] }],
    ['DataAnalyzer']
  ]
});
```

### Conditional Plugin Usage
```javascript
const result = await brain.process(data, {
  usePlugins: data.type === 'numeric' 
    ? ['DataNormalizer', 'StatisticalAnalyzer']
    : ['TextProcessor', 'NLPAnalyzer']
});
```
