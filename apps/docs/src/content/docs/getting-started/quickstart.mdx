## Basic Usage
<Steps>
1. Import the agent setup function:
   ```javascript
    import { setupAgent } from '@iqai/agent';
    import { ModelProviderName } from "@elizaos/core";
   ```
2. Create a basic agent with your OpenAI key:
   ```javascript
   const agent = new Agent({
     openAiKey: process.env.OPENAI_API_KEY
   });
   ```
   
</Steps>
## Configuration Options
<Tabs>
  <TabItem label="Basic">
    ```javascript
    const agent = await setupAgent({
      modelProvider: ModelProviderName.OPENAI,
      openAiKey: process.env.OPENAI_API_KEY
    });
    ```
  </TabItem>
  <TabItem label="With Character">

## Best Practices
### Error Handling
```javascript
try {
  const agent = await setupAgent(config);
} catch (error) {
  console.error('Failed to setup agent:', error);
}
```
### Graceful Shutdown
```javascript
process.on('SIGTERM', async () => {
  await agent.close();
  process.exit(0);
});
```
