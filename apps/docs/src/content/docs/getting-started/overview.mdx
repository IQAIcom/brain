## Introduction

Brain Framework is a powerful toolkit for building AI agents that can interact with blockchain protocols and social media platforms

## Core Concepts

<CardGrid>
  <Card title="Agent" icon="comment-alt">
    The central entity that processes inputs and manages interactions. Built using AgentBuilder with various configurations.
  </Card>

  <Card title="Clients" icon="random">
    Communication interfaces that enable interaction through different channels (Direct, Telegram, Twitter).
  </Card>

  <Card title="Plugins" icon="puzzle">
    Extensions that add specific capabilities like DeFi operations, token swaps, or scheduled tasks.
  </Card>

  <Card title="Model Providers" icon="setting">
    AI models that power the agent's understanding and responses. With support for OpenAI, Claude, and more.
  </Card>
</CardGrid>
